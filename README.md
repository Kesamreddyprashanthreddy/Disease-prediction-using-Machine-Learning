An advanced multi-disease prediction system powered by machine learning, featuring real-time analysis for **Lung Disease**, **Diabetes**, **Kidney Disease**, and **Breast Cancer**.

---

## 🌟 Key Features

### 🔬 Disease Detection Modules

- **🫁 Lung Disease Detection** - CNN-based chest X-ray analysis for pneumonia and tuberculosis
- **🩺 Diabetes Risk Assessment** - KNN classifier for Type 2 diabetes prediction (92.7% accuracy)
- **🫘 Kidney Disease Analysis** - Random Forest classifier for chronic kidney disease detection (99%+ accuracy)
- **🎗️ Breast Cancer Screening** - VGG16 transfer learning for mammography analysis with BI-RADS assessment

### ✨ Core Capabilities

- ✅ Real-time AI predictions with confidence scoring
- ✅ Professional medical reports in PDF format
- ✅ Prediction history tracking with database storage
- ✅ User authentication system with secure password hashing
- ✅ Responsive design for mobile and desktop
- ✅ Interactive visualizations using Plotly
- ✅ Multi-database support (MongoDB, PostgreSQL, MySQL, SQLite)
